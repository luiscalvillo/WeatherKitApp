## WeatherKit App
A minimalist black and white weather app using Apple's WeatherKit

## Preview
![me](https://github.com/luiscalvillo/WeatherKitApp/blob/main/Screenshots/weatherkit-iphone.gif)

## Technologies
* UIKit
* WeatherKit
* MVVM
* Full programmatic UI

## Features
* Current local weather
* Hourly forecast
* Daily forecast

## Requirements
* iOS 17+
* Swift 5
